# JobPulse Frontend

Frontend Angular pour la plateforme JobPulse.

## Prérequis

- Node.js 18+
- npm

## Installation

```bash
npm install
```

## Développement

```bash
npm start
```

L'application sera disponible sur http://localhost:4200

Le proxy redirige les requêtes `/api` vers le backend (http://localhost:8000).

## Build production

```bash
npm run build
```

Les fichiers seront générés dans `dist/jobpulse-frontend/`.
